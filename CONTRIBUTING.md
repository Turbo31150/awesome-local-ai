# Contributing to Awesome Local AI

Thanks for your interest in contributing!

## Guidelines

- **One tool per pull request** — keeps reviews simple.
- **Format**: `[Tool Name](https://link) — Short description (one line).`
- **Local/self-hosted only** — tools must be runnable without mandatory cloud dependencies.
- **No duplicates** — check the list before submitting.
- **Alphabetical order** within each category.
- **Working links** — verify your links before submitting.

## How to contribute

1. Fork this repository
2. Create a new branch (`git checkout -b add-tool-name`)
3. Add your tool to the appropriate category in `README.md`
4. Commit your changes (`git commit -m 'Add tool-name'`)
5. Push to your branch (`git push origin add-tool-name`)
6. Open a Pull Request

## Adding a new category

If your tool doesn't fit any existing category, propose a new one in your PR description.

## Quality standards

- The tool should be actively maintained
- It should have clear documentation
- It should be usable for local/edge AI workflows
